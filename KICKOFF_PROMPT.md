   React 18 + Vite в frontend/, Telegram Mini App SDK. Реализуй 5 ядровых экранов Фазы 1:

   Калькулятор, Поиск. Палитра, powerline-сегменты для статей, состояния загрузки/пустоты/
   ошибки — всё в §14, включая точные hex-цвета. Не изобретай новую эстетику — воспроизведи
   мокапы гейт-экрана и статьи, которые уже согласованы (описаны текстом в §14, если
   скриншотов при тебе нет — ориентируйся на таблицу токенов и описание сегментов).
   Deep links (§16) — startapp параметр из Telegram должен роутить на нужный экран после
   прохождения гейта.
