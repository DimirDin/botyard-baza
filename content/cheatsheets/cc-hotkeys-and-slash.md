### Хоткеи
| Хоткей | Действие |
|---|---|
| Ctrl+C | Прервать генерацию / остановить цикл |
| Ctrl+D | Завершить сессию и выйти |
| Ctrl+L | Очистить экран терминала |
| ↑ / ↓ | Листать историю вводов |
| Tab | Автодополнение путей и команд |

### Slash-команды (сверено с code.claude.com, 04.07.2026)
| Команда | Зачем | Алиасы |
|---|---|---|
| /compact [фокус] | Сжать историю диалога, освободить контекст | — |
| /clear | Начать новый диалог с чистым контекстом (память проекта сохраняется) | /reset, /new |
| /rewind | Откатить код и/или диалог к чекпоинту | /checkpoint, /undo |
| /review [PR] | Ревью GitHub pull request тем же движком, что /code-review | — |
| /code-review [--fix] | Ревью текущего diff на баги; --fix сразу применяет правки | — |
| /security-review | Глубокий разбор diff на уязвимости (инъекции, auth, утечки данных) | — |
| /feedback [report] | Отправить баг-репорт с контекстом сессии | /bug, /share |
| /context [all] | Визуализация использования контекстного окна | — |
| /model [модель] | Переключить модель, сохранить как дефолт | — |
| /effort [уровень] | low/medium/high/xhigh/max — глубина рассуждения | — |
| /mcp | Управление подключениями MCP-серверов | — |
| /permissions | Правила allow/ask/deny для инструментов | /allowed-tools |
| /init | Сгенерировать стартовый CLAUDE.md | — |
| /usage | Стоимость сессии, лимиты тарифа, разбивка по скиллам/агентам | /cost, /stats |

⚠️ Уточнение: `/bug` и `/review` в интернет-подборках промптов часто описывают неверно (как «изоляция контекста под дебаг» и «аудит текущей ветки») — реальное назначение выше.
