Сверено с code.claude.com/docs/en/permission-modes, 04.07.2026.

| Режим | Поведение |
|---|---|
| default | Спрашивает подтверждение на каждое потенциально опасное действие |
| acceptEdits | Правки файлов применяются без вопросов, остальное — как в default |
| plan | Сначала анализирует и показывает план, без правок до одобрения |
| auto | Классификатор сам решает, какие действия безопасны — без лишних вопросов |
| dontAsk | Разрешены только заранее одобренные (allowlist) инструменты |
| bypassPermissions | Полностью пропускает все проверки |

Переключение на лету — Shift+Tab циклически проходит по режимам. Задать стартовый режим — флагом `--permission-mode` или в settings.json полем `defaultMode`.

⚠️ bypassPermissions — сознательный компромисс. Используй только в изолированных окружениях (контейнер, тестовая VM), не в проектах с непроверенными зависимостями.
