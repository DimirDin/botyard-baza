### ❓ Что это
Актуальная линейка моделей Anthropic делится по балансу интеллекта, скорости и цены: Fable 5 — сверхтяжёлая модель для автономных агентов, Opus 4.8 — флагман для сложного кодинга, Sonnet 5 — универсальная рабочая лошадка, Haiku 4.5 — бюджетная модель для простых задач.

### 🎯 Зачем тебе
Верный подбор модели под шаг архитектуры бережёт бюджет и latency. Нет смысла гонять Opus или Fable для классификации текста — с этим справится Haiku за доли секунды.

### 💻 Минимальный пример
| Модель | Контекст | Input / Output за 1M токенов | Юзкейс |
|---|---|---|---|
| Claude Fable 5 | 1M | $10.00 / $50.00 | Автономные долгоживущие агенты |
| Claude Opus 4.8 | 1M | $5.00 / $25.00 | Хардкорный многошаговый кодинг |
| Claude Sonnet 5 | 1M | $3.00 / $15.00 (промо $2/$10 до 31.08.2026) | Дефолт для продакшена |
| Claude Haiku 4.5 | 200k | $1.00 / $5.00 | Классификация, real-time чаты |

### ⚠️ Грабли
У Opus 4.8 есть ускоренный Fast Mode — он поднимает цену примерно до $10/$50 за 1M токенов вместо стандартных $5/$25. Включение Fast Mode без лимитов в коде может неожиданно раздуть счёт на тяжёлых запросах — сверяй актуальные условия перед включением в продакшене.
