### ❓ Что это

Артефакты — отдельное интерактивное окно рядом с диалогом, куда Claude выносит самостоятельный
блок контента: код с живым превью (React-компонент реально рендерится, не просто показан текстом),
SVG-графику, Mermaid-диаграммы, HTML-страницы. Триггерится автоматически, когда контент достаточно
самодостаточен и объёмен, чтобы иметь смысл отдельно от текста ответа — сгенерированный React-
компонент, а не короткий сниппет на три строки.

### 🎯 Зачем тебе

Любая задача, где результат — не просто текст для чтения, а нечто, с чем нужно визуально
взаимодействовать: прототип интерфейса, который сразу можно покликать, диаграмма архитектуры,
которую удобнее видеть отрисованной, а не как код Mermaid-разметки. Артефакт живёт отдельно и
редактируется независимо от продолжения разговора — не нужно пересоздавать весь код заново, если
дальше в диалоге просишь точечную правку.

### 💻 Минимальный пример

```
Сделай интерактивный калькулятор конвертации валют на React
```

Артефакт откроется отдельным окном с реально работающим компонентом — можно вводить числа и видеть
результат сразу, не переключаясь на внешний редактор кода для проверки.

### ⚠️ Грабли

- **Не для любого кода автоматически** — короткие сниппеты и объяснения кода остаются в обычном тексте
  ответа, артефакт триггерится для достаточно самодостаточного и объёмного контента.
- **Изменения в артефакте по ходу разговора создают версии** — предыдущие варианты не теряются
  бесследно, но по умолчанию виден последний, для сравнения версий нужно явно к ним обращаться.
- **Песочница артефакта ограничена** — не любая библиотека или сетевой запрос доступны внутри
  артефакта, для продакшен-разработки это прототип/демонстрация, не замена полноценного окружения
  разработки.
