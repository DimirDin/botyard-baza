### ❓ Что это

Поиск по прошлым чатам — Claude может находить и подтягивать релевантный контекст из прежних
разговоров пользователя, не требуя от него пересказывать заново, что обсуждалось раньше. Это
отдельная функция от долгосрочной Memory (которая хранит извлечённые факты) — chat search работает
именно с историей самих разговоров как с источником поиска, находя конкретные прошлые обсуждения по
смыслу запроса.

### 🎯 Зачем тебе

Продолжение работы над темой, начатой недели назад: «мы это уже обсуждали, найди тот разговор про
архитектуру платёжного модуля и продолжи оттуда» — вместо ручного пролистывания истории чатов или
полного пересказа контекста заново, Claude сам находит релевантный прошлый разговор и опирается на
него.

### 💻 Минимальный пример

```
Помнишь, мы обсуждали миграцию на JWT пару недель назад? Продолжи с того места
```

Claude ищет по истории чатов пользователя, находит релевантный разговор, подтягивает его содержание
как контекст для текущего ответа — без явной ссылки на конкретную дату или название чата от
пользователя.

### ⚠️ Грабли

- **Поиск по смыслу, не точному совпадению слов** — иногда полезно уточнить формулировку, если
  первая попытка не находит нужный разговор, как с любым семантическим поиском.
- **Не заменяет структурированную Memory** — chat search находит разговоры целиком, Memory хранит
  извлечённые устойчивые факты; для «Claude должен помнить это всегда» лучше подходит явное сохранение
  в Memory, а не расчёт на то, что поиск каждый раз найдёт нужный старый чат.
- **Зависит от настроек приватности/сохранения истории** — если история чатов не сохраняется по
  настройкам аккаунта, искать по ней нечего.
