### ❓ Что это

Claude может генерировать настоящие файлы прямо в чате Claude.ai — Excel-таблицы (`.xlsx`) с
формулами и форматированием, презентации (`.pptx`), PDF-документы, а не только текстовые ответы или
код-блоки. Файл создаётся как реальный документ, который скачивается и открывается в
соответствующем приложении (Excel, PowerPoint) со всей структурой — листами, формулами, стилями
слайдов — а не просто текстовым содержимым, обёрнутым в нужное расширение.

### 🎯 Зачем тебе

Задачи, финальный результат которых по своей природе — не текст, а структурированный документ:
свод бюджета с формулами суммирования по категориям, презентация для встречи с готовой структурой
слайдов, отчёт в PDF для отправки клиенту. До этой фичи такие задачи требовали ручного переноса
текстового ответа Claude в Excel/PowerPoint самим пользователем — сейчас файл выходит готовым к
использованию.

### 💻 Минимальный пример

```
Сделай Excel-таблицу с бюджетом на квартал: колонки — категория, план, факт, отклонение,
с формулой автоматического расчёта отклонения и итоговой строкой
```

Claude создаёт файл с реальными формулами Excel (не текстовыми значениями результата) — открыв файл,
можно менять числа в плане/факте и видеть, как пересчитывается отклонение и итог, как в обычной
таблице, сделанной человеком.

### ⚠️ Грабли

- **Сложность документа ограничена практическим разумием запроса** — презентация на 200 слайдов с
  уникальным дизайном каждого не то, для чего эта фича оптимальна, для больших сложных проектов
  лучше начинать со скелета и итерировать.
- **Формулы Excel генерируются моделью и могут содержать логическую ошибку** — как и любой
  сгенерированный код, проверяй финальные значения на реальных данных, а не доверяй слепо только
  потому что «выглядит как настоящая формула».
- **Не все форматы одинаково зрело поддержаны** — сверяйся с актуальным списком поддерживаемых типов
  файлов, фича развивается и список может расширяться.
