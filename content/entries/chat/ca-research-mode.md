### ❓ Что это
Research — режим глубокого поиска: вместо одного быстрого запроса модель итеративно ищет зацепки, уточняет запросы, сопоставляет источники и формирует комплексный отчёт со ссылками.

### 🎯 Зачем тебе
Хорош для ресерча, когда готового ответа нет на первой странице поисковика — например, сравнение условий разных юрисдикций или бенчмарк производительности новых версий инструментов.

### 💻 Минимальный пример
Запрос «Проанализируй последние изменения в API X за этот год» запускает цепочку: поиск → парсинг → уточняющий поиск → кросс-проверка → синтез с источниками.

### ⚠️ Грабли
Это медленный режим — ответ может занимать минуты, и расходует лимит сообщений быстрее обычного чата. Включай осознанно, для комплексных исследований.
