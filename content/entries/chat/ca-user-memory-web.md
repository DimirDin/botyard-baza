### ❓ Что это
Memory — сквозная персонализация внутри Claude.ai: модель извлекает и запоминает ключевые факты о пользователе и переносит контекст между изолированными диалогами.

### 🎯 Зачем тебе
Убирает необходимость повторного онбординга модели в каждом новом чате — не нужно заново объяснять свой стек и текущие проекты.

### 💻 Минимальный пример
Упоминание в одном чате деталей проекта формирует запись в Settings → Memory, которая автоматически используется в следующих, не связанных чатах.

### ⚠️ Грабли
Память забивается устаревшим контекстом. Периодически заходи в Settings → Personalization → Memory и вручную вычищай неактуальные факты.
