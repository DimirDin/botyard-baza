### ❓ Что это
Claude Code хранит логи диалогов, метаданные изменённых файлов и индекс репозитория в локальном хранилище на машине пользователя. Это позволяет продолжать сессию после закрытия терминала.

### 🎯 Зачем тебе
Не нужно каждый раз заново объяснять контекст — можно закрыть ноутбук и вернуться на следующий день ровно с того места, где остановился, без пересканирования диска.

### 💻 Минимальный пример
```bash
cc --list-sessions              # список сохранённых сессий
cc --session-id=sess_01j1abc123 # восстановить конкретную сессию
cc --clear-cache                # сбросить локальный кэш и индекс
```

### ⚠️ Грабли
История хранится локально в открытом виде. Если в чат случайно попали секреты или коммерческие данные, они останутся в этом хранилище — делай регулярную чистку сессий на рабочих машинах с жёстким SLA по данным.
