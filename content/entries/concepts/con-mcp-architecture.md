### ❓ Что это
MCP — открытый протокол клиент-серверной связи на базе JSON-RPC 2.0. MCP-сервер анонсирует свои инструменты и ресурсы, а любой MCP-клиент (Claude Code, Claude Desktop) может их использовать без написания уникальной интеграции.

### 🎯 Зачем тебе
Решает проблему фрагментации инструментов. Готовый MCP-сервер для Postgres даёт модели строго типизированные инструменты для чтения таблиц без написания кастомного плагина.

### 💻 Минимальный пример
```json
{ "jsonrpc": "2.0", "method": "tools/list", "id": 1 }
```
Транспорт — stdio или SSE между клиентом (LLM-хост) и сервером (реальный код, дёргающий внешний софт).

### ⚠️ Грабли
MCP-сервер — реальный исполняемый код на твоей машине. Скачивание непроверенных серверов может дать постороннему коду доступ к файловой системе. Всегда проверяй исходники кастомных серверов перед добавлением в конфиг.
