### ❓ Что это

У артефакта три пути наружу: скачать как отдельный файл, опубликовать по постоянной ссылке (её
может открыть человек без аккаунта Claude — просто в браузере), либо в Claude Code — опубликовать
прямо из сессии на приватный URL на claude.ai, который обновляется по ходу работы.

### 🎯 Зачем тебе

Ссылка на артефакт — быстрый способ показать клиенту прототип без «зайди в мой чат». Экспорт из
Claude Code как отдельная страница удобен для вещей вроде разбора PR с аннотированным диффом или
дашборда по логам, которые команда должна видеть обновляющимися по ходу расследования, а не как
статичный скриншот.

### 💻 Минимальный пример

В Claude Code: «Сделай артефакт с дашбордом ошибок деплоя за неделю по сервисам и обнови его по
ходу расследования» — Claude Code спросит разрешение перед публикацией и даст приватную ссылку.

### ⚠️ Грабли

Опубликованные артефакты выставляют счёт за вычисления конечному пользователю, который их
использует, а не автору — это удобно для шаринга без роста собственных расходов, но значит, что
у зрителя должен быть свой доступ к Claude, если артефакт делает вызовы к API. Публикация в Claude
Code требует подтверждения перед каждым разом — по умолчанию ничего не публикуется молча.
