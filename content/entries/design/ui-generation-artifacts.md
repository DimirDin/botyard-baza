### ❓ Что это

Artifacts — режим, в котором Claude выносит существенный кусок контента (код, документ, интерфейс)
в отдельное окно рядом с чатом, вместо того чтобы вставлять всё в переписку. Когда просишь
интерактивную вещь — форму, калькулятор, дашборд — Claude пишет HTML/CSS/JS или React-компонент,
и это сразу рендерится живым интерфейсом в панели артефакта, а не просто лежит текстом.

### 🎯 Зачем тебе

Не нужно копировать код в отдельный редактор, чтобы посмотреть, что получилось — результат виден
сразу, и по нему можно итерировать репликами в духе «сделай кнопки крупнее» или «добавь тёмную
тему». Годится для прототипов, внутренних утилит, визуализаций данных, обучающих демо — везде,
где интерфейс проще показать, чем описать.

### 💻 Минимальный пример

Промпт: «Сделай калькулятор чаевых: сумма счёта, процент чаевых ползунком, количество человек,
итог на человека» — Claude сгенерирует рабочий HTML/JS-виджет прямо в артефакте, без вопросов о
стеке.

### ⚠️ Грабли

Генеративный UI работает целиком на стороне клиента в песочнице — обычный артефакт не может делать
серверные вызовы к внешним API, не сохраняет данные между сессиями и не подключается к настоящей
базе данных. Для персистентности и вызовов API из самого артефакта нужны отдельные механизмы
(персональное хранилище артефактов, AI-powered artifacts с доступом к Claude API) — это не
включено по умолчанию в обычный артефакт.
