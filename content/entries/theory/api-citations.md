### ❓ Что это

Citations — режим, в котором документы передаются как структурированный контент, и Claude возвращает
ответ с точными ссылками на конкретные предложения/страницы источника для каждого утверждения. В
ответе это отдельные поля `citations` внутри текстового блока — системная привязка на уровне API
между сгенерированным текстом и исходным фрагментом, а не ссылка «по памяти».

Документы подаются с `"citations": {"enabled": true}` в блоке документа — PDF, plain text или готовые
чанки.

### 🎯 Зачем тебе

Сценарии, где важно проверить источник факта: юридический и медицинский Q&A по документам, саммари
контрактов с точными пунктами, support-бот по базе знаний. Citations снимают классическую проблему
RAG-ботов «уверенный ответ оказался галлюцинацией» — оставляют пользователю способ проверить
одним кликом.

Ключевое отличие от обычного «попроси модель указать источник в тексте ответа» — системная
гарантированность привязки. Текстовая просьба «укажи, откуда этот факт» полагается на то, что модель
сама честно и точно процитирует нужный фрагмент — она может перефразировать, обобщить или указать
неточный участок. Citations работают на уровне API как структурная связь между сгенерированным
сегментом текста и конкретными символами исходного документа, а не как текстовая договорённость,
которую модель может интерпретировать вольно.

### 💻 Минимальный пример

```json
{
  "messages": [{
    "role": "user",
    "content": [
      {
        "type": "document",
        "source": { "type": "text", "media_type": "text/plain", "data": "Полный текст договора..." },
        "citations": { "enabled": true }
      },
      { "type": "text", "text": "Какой срок действия договора и есть ли автопродление?" }
    ]
  }]
}
```

Ответ содержит текстовые сегменты вперемешку с объектами вида
`{"type": "char_location", "document_index": 0, "start_char_index": 120, "end_char_index": 180}`.

### ⚠️ Грабли

- **Работает только с `document`-блоком с `citations.enabled`** — просто вставленный текст в промпт
  привязки не даёт.
- **Не отменяет качество чанкинга** — плохо порезанный документ даёт технически верную, но вырванную
  из контекста цитату.
- **Дороже по токенам** — в вывод добавляются метаданные локаций.
- **Не путай с web search tool** — там источники внешние, здесь — твои загруженные документы.
