### ❓ Что это

Линейка моделей Claude 5 — Haiku (быстрая и дешёвая, для высокочастотных простых подзадач), Sonnet
(баланс цены/качества, дефолт для большинства продуктов), Opus (максимум для самых сложных задач),
плюс Fable отдельной линейкой. Каждая новая генерация смещает границы: что раньше требовало Opus,
через одну-две генерации часто закрывается уровнем Sonnet.

### 🎯 Зачем тебе

Проектирование пайплайна — не одна модель на всё, а распределение ролей: Haiku на классификацию,
Sonnet как рабочая лошадка, Opus точечно там, где нужна глубина.

### 💻 Минимальный пример

| Шаг | Модель | Почему |
|---|---|---|
| Классификация запроса | Haiku | простая задача, высокая частота |
| Извлечение сущностей | Haiku/Sonnet | средняя сложность |
| Финальный синтез | Sonnet | нужно качество |
| Код-ревью на скрытый баг | Opus | максимум рассуждения |

### ⚠️ Грабли

- **Не выбирай модель по общей репутации** — сверяйся с бенчмарками под свой тип задачи.
- **Цена растёт нелинейно между уровнями** — планируй капасити по реальному прайсингу.
- **Модели меняются со временем** — фиксируй конкретную версию в проде, если критична
  предсказуемость.
