### ❓ Что это

Поддержка PDF работает как гибридный анализ: Claude получает и извлечённый текст, и визуальное
представление каждой страницы. Модель видит таблицы, диаграммы, подписи на графиках, рукописные
пометки — всё, что теряется при чистом текстовом извлечении, — и опирается на точный текст там, где
он важен (даты, суммы, номера пунктов).

Передаётся как `document`-блок с `media_type: "application/pdf"` и base64, либо через Files API по
`file_id` — выгоднее, если один PDF используется в нескольких запросах.

### 🎯 Зачем тебе

Разбор счетов и накладных со сканами таблиц, анализ статей с графиками, извлечение данных из
презентаций, ревью договоров со сложной вёрсткой. Чистый текстовый экстрактор часто ломает порядок
колонок таблиц — vision-компонент это компенсирует.

Гибридный подход решает конкретный класс проблем, знакомый любому, кто пробовал автоматически
распарсить PDF: текстовый слой документа хранит символы в порядке их размещения в файле, который
далеко не всегда совпадает с визуальным порядком чтения — многоколоночная вёрстка, таблицы с
объединёнными ячейками, подписи к диаграммам легко превращаются в кашу при чистом текстовом
извлечении. Подключение зрения к тому же документу даёт модели способ сверяться с тем, как страница
выглядит на самом деле, а не только с тем, что формально записано в текстовом потоке файла.

### 💻 Минимальный пример

```json
{
  "messages": [{
    "role": "user",
    "content": [
      { "type": "document", "source": { "type": "base64", "media_type": "application/pdf", "data": "<base64>" } },
      { "type": "text", "text": "Собери таблицу расходов по месяцам из этого PDF" }
    ]
  }]
}
```

Через Files API: `{ "type": "document", "source": { "type": "file", "file_id": "file_abc123" } }`.

### ⚠️ Грабли

- **Лимит по числу страниц и размеру файла** — длинные PDF режь на части.
- **Каждая страница считается токенами примерно как изображение** — PDF на 40 страниц съедает
  контекст быстрее, чем кажется по весу файла.
- **Сканы низкого качества** снижают точность так же, как снижали бы у человека.
- **Файлы с паролем не читаются** — разблокируй заранее.
