### ❓ Что это

Vision API — анализ изображений (скриншоты, фото, макеты, диаграммы) через `image`-блок с base64 или
Files API по `file_id`. Токены за изображение считаются по формуле, зависящей от разрешения — модель
сама читает текст на изображении как часть визуального анализа, без отдельной OCR-прогонки заранее.

### 🎯 Зачем тебе

Ревью UI-макетов и скриншотов, анализ диаграмм и графиков без структурированных данных под ними,
распознавание рукописных заметок, сравнение до/после дизайна.

### 💻 Минимальный пример

```json
{
  "messages": [{
    "role": "user",
    "content": [
      { "type": "image", "source": { "type": "base64", "media_type": "image/png", "data": "<base64>" } },
      { "type": "text", "text": "Что не так с версткой этой кнопки?" }
    ]
  }]
}
```

### ⚠️ Грабли

- **Лимит по весу и разрешению** — очень большие изображения масштабируются автоматически или
  отклоняются.
- **Токены за картинку считаются не «плоско»** — оптимизируй разрешение под задачу заранее.
- **Точность на мелком/повёрнутом тексте ограничена** — как и у человека, для критичных данных
  дублируй текстовым источником.
