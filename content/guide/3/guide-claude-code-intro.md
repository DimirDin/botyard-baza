### ❓ Что это
**Claude Code** — официальный агентный CLI-инструмент от Anthropic. В отличие от claude.ai, он работает прямо в терминале, видит файлы твоего проекта, может сам их читать, менять, запускать команды и коммитить в git.

### 🎯 Зачем тебе
Если ты пишешь код, Claude Code избавляет от постоянного копирования кусков туда-сюда между чатом в браузере и редактором. Ты просто говоришь ему, что сделать — он сам находит нужные файлы, правит их и показывает диф.

### 💻 Как это выглядит на практике
Установка одной командой:
```bash
curl -fsSL https://claude.ai/install.sh | bash
```
Дальше в терминале, находясь в папке проекта:
```
cd /путь/к/проекту
claude
```
И общаешься с ним так же текстом, но результат — реальные изменения файлов в этой папке, а не просто ответ в чате.

### ⚠️ Частая ошибка новичка
Давать агенту слишком общую задачу без структуры проекта («сделай сайт лучше») — Claude Code работает намного точнее, если у него есть конкретика: какой файл, что именно поменять, какой результат ожидается. Об этом — следующий урок, про CLAUDE.md.
