### ❓ Что это
**MCP** (Model Context Protocol) — открытый стандарт связи между Claude и внешними инструментами. MCP-сервер «объявляет», какие у него есть инструменты (например, «прочитать таблицу из базы данных»), а Claude может их вызывать, не требуя отдельной кастомной интеграции под каждый сервис.

### 🎯 Зачем тебе
Без MCP каждую интеграцию пришлось бы писать с нуля. С готовым MCP-сервером, например, для Postgres, Claude сразу получает типизированные инструменты для чтения таблиц — установил и подключил, а не написал плагин самостоятельно.

### 💻 Как это выглядит на практике
Подключение MCP-сервера в Claude Code — одной командой:
```bash
claude mcp add имя-сервера -- npx -y <mcp-пакет>
claude mcp list   # проверить, что подключилось
```
После подключения в диалоге можно просто попросить: «покажи структуру таблицы users» — и Claude сам вызовет нужный инструмент MCP-сервера.

### ⚠️ Частая ошибка новичка
Подключать непроверенные MCP-серверы из случайных источников. MCP-сервер — это реальный исполняемый код на твоей машине, который может иметь доступ к файлам или сети. Проверяй исходники перед подключением, особенно если сервер не от известного разработчика.
