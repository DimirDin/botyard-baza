### ❓ Что это
**Headless-режим** запускает Claude Code без интерактивного диалога: задача передаётся аргументом командной строки, агент отрабатывает автономно и возвращает код завершения — 0 при успехе, 1 при ошибке.

### 🎯 Зачем тебе
Это позволяет встроить агента в пайплайн CI/CD — на каждый Pull Request автоматически запускается ревью кода или генерация недостающих тестов, ещё до того, как к делу подключится человек.

### 💻 Как это выглядит на практике
Шаг в GitHub Actions:
```yaml
- uses: actions/checkout@v4
- run: npm install -g @anthropic-ai/claude-code
- env:
    ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
  run: claude-code --non-interactive "Проверь diff, найди критичные баги, верни код 1 если есть"
```
Каждый PR автоматически прогоняется через агента без ручного запуска.

### ⚠️ Частая ошибка новичка
Не ограничивать бюджет токенов и таймаут шага. Если сборка циклически падает, headless-агент может бесконечно пытаться её исправить и выжечь весь баланс API за считанные минуты — всегда ставь лимиты.
